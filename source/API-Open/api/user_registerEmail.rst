
2.2 用户邮箱注册
~~~~~~~~~~~~~~~~~~~~~~~~

:说明: 注册成为钱包的用户。
:接口地址: api/user/registerEmail
:请求方式: POST
:请求参数:

======= ======= ======== =================================================
param	type	是否必须	说明
email	string	必填	邮箱
app_id	string	必填	商户的唯一标识
time	long	必填	当前时间戳
sign	string	必填	签名
======= ======= ======== =================================================

:响应参数:

======= ======= ======== =================================================
param	type	是否必须	说明
uid	int	必填	用户在钱包服务的唯一标识
======= ======= ======== =================================================

:响应示例:

::

	{
	    "code": "0",
	    "msg": "suc",
	    "data": {
	        "uid"：10001
	    }
	}
