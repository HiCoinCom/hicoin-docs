
2.5 获取用户所有币账户
~~~~~~~~~~~~~~~~~~~~~~~~

:说明: 根据用户ID查询用户所有币种的账户
:接口地址: /api/account/getAllAccount
:请求方式: GET
:请求参数:

======= ======= ======== =================================================
param	type	是否必须	说明
uid	string	必填	用户ID
app_id	string	必填	商户的唯一标识
time	long	必填	时间戳
sign	string	必填	签名
======= ======= ======== =================================================

:响应参数:

================ ======= ======== =================================================
param            type    是否必须  说明
symbol           string  必填      币种名称
normal_balance   string  必填      正常余额
lock_balance     string  必填      冻结余额
================ ======= ======== =================================================

:响应示例:

::

	{
	  "code": "0",
	  "msg": "suc",
	  "data":[
	      {
	        "symbol": "BTC",
	        "normal_balance": "1.00211211",
	        "lock_balance": "0.00211002"
	      },
	      {
	        "symbol": "ETH",
	        "normal_balance": "1.00211211",
	        "lock_balance": "0.00211002"
	      }
	  ]
	}
